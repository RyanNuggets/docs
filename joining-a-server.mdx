<Info>
  Join codes are provided by your server's staff. If you don't have one yet, ask a moderator or administrator in your community.
</Info>

<Steps>
  <Step title="Log in with Discord">
    Open the dashboard and authenticate with your Discord account if you haven't already.
  </Step>
  <Step title="Navigate to Join Server">
    From your dashboard, locate and click the **"Join Server"** button.
  </Step>
  <Step title="Enter your join code">
    Paste or type the join code provided by your server's staff into the input field.
  </Step>
  <Step title="Confirm and join">
    Click **Join** to complete the process. The server will immediately appear on your dashboard.
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Join code isn't working">
    Join codes may have a maximum use limit set by the server owner. If your code has expired or reached its limit:

    - Contact your server's staff or moderators for a new code
    - Verify you've entered the code correctly (codes are case-sensitive)
    - Check that you're not already a member of the server
  </Accordion>

  <Accordion title="Server doesn't appear after joining">
    If the server doesn't show up on your dashboard immediately:

    - Refresh your browser or reload the dashboard
    - Check that you're logged in to the correct Discord account
    - Clear your browser cache and try again
  </Accordion>

  <Accordion title="I don't have a join code">
    Ask a server moderator or administrator for an invite code. They can generate a new one with custom expiration and usage limits.
  </Accordion>
</AccordionGroup>

<Tip>
  Save your join code in a safe place if you plan to invite others, or ask your server staff to generate a shareable link for your community.
</Tip>
