The **Department of Transportation (DOT) CAD** handles traffic and road operations for your community. DOT units manage incident calls, log records, and track active road closures across the map.

## Main Features

<Columns cols={2}>
  <Card title="Main" icon="dashboard">
    Active calls board, your assigned calls, live units, and a map preview
  </Card>

  <Card title="Map" icon="map">
    Full-screen live player map (requires ERLC API key)
  </Card>

  <Card title="Calls" icon="phone">
    Full calls list across all departments with filtering
  </Card>

  <Card title="Records" icon="file-lines">
    Incident reports, traffic citations, BOLOs, and warrants
  </Card>

  <Card title="Roads" icon="road">
    Active road closure tracking board
  </Card>
</Columns>

## Permissions

<Info>
  Users need the **"View DOT"** permission to access the DOT CAD. Managing records or operations requires the **"Manage DOT"** permission.
</Info>
