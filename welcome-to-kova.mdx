Kova Hub is a **free, enterprise-grade CAD (Computer Aided Dispatch) system** purpose-built for ER:LC (Emergency Response: Liberty County) communities on Roblox. Whether you're managing a small private server or scaling a large regional community, Kova Hub delivers professional dispatch infrastructure without the cost.

## Core Features

<AccordionGroup>
  <Accordion title="Dispatch & Call Management" defaultOpen>
    - Real-time dispatch board with live call tracking
    - Automatic unit assignment and status updates
    - Call history and incident archiving
    - Multi-channel dispatch routing
  </Accordion>

  <Accordion title="Police Operations">
    - Civilian database with lookup tools
    - Warrant and BOLO management
    - Arrest records and booking system
    - Traffic stop documentation
  </Accordion>

  <Accordion title="Fire & EMS">
    - Incident response tracking
    - Crew roster and certification management
    - Medical call records
    - Response time analytics
  </Accordion>

  <Accordion title="Administrative Tools">
    - Role-based access control (RBAC)
    - Unit groups and team management
    - Custom ten codes and penal codes
    - Audit logging and compliance tracking
  </Accordion>

  <Accordion title="Integration & API">
    - Discord bot for real-time notifications
    - ERLC API for live map and bodycam
    - Webhook support for custom workflows
    - Data export and reporting
  </Accordion>
</AccordionGroup>
