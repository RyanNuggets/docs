<Info>
  Connecting your Roblox ER:LC private server unlocks the live map, which displays player positions in real time on your CAD's map panel.
</Info>

<Steps>
  <Step title="Open Server Management">
    Navigate to **Manage Server** → **Integrations** → **Roblox Integration**
  </Step>
  <Step title="Locate Your API Key">
    Find your API key in your ER:LC private server settings within Roblox
  </Step>
  <Step title="Paste and Save">
    Paste the key into the **ERLC Server API Key** field and click **Save Key**
  </Step>
</Steps>

## Remove Your API Key

To disconnect the integration, click **Remove Key**. The live map will stop updating immediately.

<Warning>
  Without an active ERLC API key, the live map will not populate and 911 webhooks will not function.
</Warning>
