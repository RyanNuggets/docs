---
title: "Roles & Permissions"
---

Kova Hub has a fully granular role-based permission system. You create custom roles and assign any combination of permissions to them. Members can hold multiple roles.

**Permission Groups:**

Server

- Super Admin — Full server management access except the danger zone
- Manage Server — Edit general server configuration
- Manage Roles — Create and update roles
- Manage Departments — Create and update departments and divisions
- Manage Integrations — Configure Discord and external integrations

Members

- View Members — See server members and their assigned roles
- Manage Members — Edit member roles and remove access
- Invite Members — Invite users into the server

Records

- View Records — View reports and department records
- Create Records — Create new incident and case records
- Edit Records — Update existing records
- Delete Records — Remove records permanently

Dispatch

- View Dispatch — Access the dispatch board
- Manage Dispatch — Edit calls, statuses, and assignments

Supervisor

- View Bodycam — View live bodycam feeds from other units without requiring dispatch access

Police

- View Police — Access police CAD and incident records
- Manage Police — Create and update police records

Fire / EMS

- View Fire / Manage Fire
- View Fire Roster / Manage Fire Roster
- View EMS / Manage EMS
- View EMS Roster / Manage EMS Roster

DOT

- View DOT / Manage DOT
- Issue Citations
- Manage Permits

**Note:** The server owner always has full access regardless of assigned roles.
