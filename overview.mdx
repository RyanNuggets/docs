The Dispatch board is the central hub for coordinating all active units and calls across your server. It's designed for dedicated dispatchers, though units can also self-dispatch.

## Board Panels

The board is organized into four main panels:

<Columns cols={2}>
  <Card title="Calls" icon="phone">
    All active calls across Police, Fire, EMS, and DOT departments in one view.
  </Card>

  <Card title="Units" icon="users">
    All units currently signed into the CAD, grouped by department for easy organization.
  </Card>

  <Card title="Map" icon="map">
    Live player positions from your ERLC server _(requires API key)_.
  </Card>

  <Card title="Filtering" icon="filter">
    Filter the board by department (Police, Fire, EMS, DOT) to focus on specific operations.
  </Card>
</Columns>

## Key Features

- **Multi-department coordination**: Manage calls and units across all emergency services simultaneously
- **Real-time updates**: Live unit positions and call status on an integrated map
- **Department filtering**: Isolate specific departments for focused dispatch operations
- **Flexible dispatch**: Support for both dedicated dispatchers and self-dispatching units
