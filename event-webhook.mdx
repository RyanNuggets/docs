---
title: "Event Webhook"
---

The PRC Event Webhook lets Kova Hub receive live in-game events from your ER:LC server, including 911 calls placed by civilians in-game.

When a civilian dials 911 in-game, the call appears on the dispatch board in real time with an alert chime. Dispatchers can then assign units without needing a separate system.

**Requirements:**

- Your ERLC Server API Key must be saved first (see ERLC API Key)

**Setup:** The webhook URL is generated automatically once your ERLC key is saved. Follow the on-screen setup instructions in Manage Server → Integrations → PRC Event Webhook to configure it inside your ER:LC private server settings.
