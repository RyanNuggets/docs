When a civilian dials 911 in-game, the call appears on your dispatch board instantly with an alert chime. Dispatchers can assign units without switching systems.

## Requirements

<Info>
  Before setting up the Event Webhook, ensure you have:

  - Your **ERLC Server API Key** saved in Kova Hub
  - Access to your **ER:LC private server settings**
  - **Dispatcher permissions** to manage the integration
</Info>

## Getting Started

<Steps>
  <Step title="Save Your ERLC Server API Key">
    Ensure your ER:LC Server API Key is configured in your account settings. [Learn how to set up your API key](/erlc-api-key).
  </Step>
  <Step title="Generate Webhook URL">
    Your webhook URL is generated automatically once your ERLC key is saved. No manual configuration needed.
  </Step>
  <Step title="Configure in ER:LC Settings">
    Go to **Manage Server → Integrations → PRC Event Webhook** and paste the generated URL into your ER:LC private server settings.
  </Step>
  <Step title="Start Receiving Events">
    Once configured, live 911 calls will appear on your dispatch board in real time.
  </Step>
</Steps>

<Callout icon="question" color="#8B5CF6">
  If you encounter issues during setup, check your API key configuration or contact support for assistance.
</Callout>
