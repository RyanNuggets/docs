Unit groups let you organize units into named teams during active operations (e.g. "Traffic Stop Alpha," "Fire Scene Unit 2"). Groups appear on the dispatch board so dispatchers can see which units are working together.

## Key Operations

<Columns cols={3}>
  <Card title="Creating a Group" icon="plus">
    Any unit can create a group from the CAD. Other units can be invited to join by other units.
  </Card>

  <Card title="Joining a Group" icon="user-plus">
    Units receive an invite notification and can accept to join. Units can only be in one group at a time.
  </Card>

  <Card title="Leaving a Group" icon="log-out">
    Units can leave a group at any time. The group is automatically removed when it's empty.
  </Card>
</Columns>
