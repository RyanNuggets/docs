Server owners and users with the appropriate permissions can access the Manage Server panel. It's split into the following tabs:

<Columns cols={2}>
  <Card title="General" icon="sliders" href="#general">
    Configure server name, description, icon, join codes, ten codes, penal codes, certifications, and CAD settings.
  </Card>

  <Card title="Members" icon="users" href="#members">
    View and manage all members in the server with granular control, as well as their roles and join codes.
  </Card>

  <Card title="Roles & Permissions" icon="shield" href="#roles-permissions">
    Create custom roles and assign granular permissions to control access. 
  </Card>

  <Card title="Departments & Divisions" icon="sitemap" href="#departments-divisions">
    Organize your community into departments with hierarchical sub-divisions.
  </Card>

  <Card title="Integrations" icon="plug" href="#integrations">
    Connect Discord and your ERLC server for seamless cross-platform management.
  </Card>

  <Card title="Danger Zone" color="#DC2626" icon="triangle-exclamation" href="#danger-zone">
    Irreversible actions such as deleting the server, handle with extreme caution when on this tab.
  </Card>
</Columns>

<Info>
  Only server owners and users with the appropriate permissions can access the Manage Server panel. Check your role settings to confirm your access level.
</Info>
