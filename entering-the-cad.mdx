Before accessing the full CAD, you'll complete an entry screen where you set your session details. Once signed in, you appear as an active unit on the dispatch board and within the CAD.

## Entry Fields

<Columns cols={2}>
  <Card title="Name" icon="user">
    Your in-game character name or Roblox username (auto-filled if linked).
  </Card>

  <Card title="Callsign" icon="id-card">
    Your unit identifier (e.g., 1-A-12).
  </Card>

  <Card title="Department" icon="building">
    Select your active department.
  </Card>

  <Card title="Division" icon="sitemap">
    Select your active division, if applicable.
  </Card>

  <Card title="Certifications" icon="certificate">
    Select any certifications active for this session (e.g., K9, FTO).
  </Card>
</Columns>

## Getting Started

Complete all required entry fields, then confirm to appear as an active unit on the dispatch board.
